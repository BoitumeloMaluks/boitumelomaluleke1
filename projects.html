Alarm Clock Project



Alarm Clock Project


This Python-based Alarm Clock application is designed with a user-friendly Graphical User Interface (GUI) using Tkinter. The program offers several functionalities, including the display of the current time in real-time. Users can conveniently set alarms by choosing specific hours and minutes from dropdown menus provided within the interface. It allows users to activate or deactivate alarms at their convenience. When the set alarm time aligns with the current time, the application triggers an alarm sound to notify the user. Additionally, a countdown timer displays the time remaining until the alarm goes off, enhancing user awareness. The application also includes a snooze feature, enabling users to temporarily pause the alarm by specifying a snooze duration, which is visualized through a countdown timer.

Point of Sale Application


Point of Sale Application


This Python script models a restaurant system allowing waiters to manage tables, take orders, prepare bills, complete sales, and perform cash-up operations. Waiters can log in, assign tables, change the number of customers at a table, add items to orders, prepare bills, complete sales, and conduct cash-up at the end of their shift. It manages tables, orders, and sales, storing data in text files and providing an interactive console-based interface for restaurant operations.
